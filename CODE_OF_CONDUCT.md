# Code of Conduct

## Our Pledge

为了构建一个开放、友好、尊重协作的社区环境，我们承诺：

- 尊重不同背景与经验的贡献者
- 以建设性的方式讨论技术问题
- 避免人身攻击、侮辱、歧视和恶意挑衅

## Expected Behavior

- 聚焦问题本身，而非个人
- 接受反馈并进行理性讨论
- 在提交 Issue / PR 时提供清晰上下文
- 尊重项目的非商业许可边界

## Unacceptable Behavior

- 骚扰、辱骂、威胁、歧视性表达
- 恶意刷 Issue / PR
- 明知违反许可仍推动商业滥用

## Enforcement

项目维护者有权对不当行为采取警告、限制讨论、关闭互动或拒绝贡献等措施。
